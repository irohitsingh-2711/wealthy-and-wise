# Wealthy and Wise

Next.js conversion of the Wealthy and Wise calculator suite.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Included tools

- FIRE Calculator
- Coast FIRE Calculator
- Rent vs Buy Calculator
- Goal SIP Calculator
- SIP Calculator
- SIP Duration Estimator

## Deploy

The app is ready for GitHub and can be deployed on Vercel after pushing the repository.

1. Create a new GitHub repository.
2. Push this project to the default branch.
3. Import the repository into Vercel.
4. Deploy with the default Next.js settings.
